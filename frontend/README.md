# Frontend

LiveCaptions 前端应用，使用 React + Vite + Tailwind CSS 构建。

## 开发命令

```bash
# 安装依赖
npm install

# 代码格式化
npm run format
npm run format:check

# 代码检查
npm run lint

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 目录结构

- `src/components/` - UI 组件
  - `ui/` - 基础 UI 组件
  - `layout/` - 布局组件
- `src/pages/` - 页面组件
- `src/contexts/` - React Context
- `src/lib/` - 工具库（API 客户端、工具函数）
